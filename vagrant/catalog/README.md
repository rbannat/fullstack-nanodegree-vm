Udacity FSND Catalog Project
=============

- install vagrant
- cd into `/vagrant`
- run `vagrant up`
- log in via `vagrant ssh`
- cd into `/vagrant/catalog/`
- create database by running `python database_setup.py`
- prefill database by running `python lotsofcategories.py`
- run `project.py`
